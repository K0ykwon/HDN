# Prompt 01 — Bootstrap the Repo

Implement the initial repository skeleton for TWR-LM in PyTorch.

## Do now
- Create the repo layout described in `AGENTS.md`.
- Add `pyproject.toml` or `requirements.txt`.
- Add a short `README.md` with setup and one training command placeholder.
- Create placeholder modules and `__init__.py` files.

## Constraints
- Keep the code minimal and clean.
- Do not implement every feature yet.
- Make the structure ready for fast iteration.

## Deliverable
