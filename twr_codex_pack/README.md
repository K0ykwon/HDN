# TWR-LM Codex Pack

This folder is a historical prompt pack for the original `Tokenless Write-Think-Read` implementation plan.

## Status

- It is kept for reference.
- It does not describe the current mainline repository faithfully.
- The live codebase has moved to a lean `tokenless latent backbone` centered on:
  - overlapping latent encoding
  - hierarchical latent refinement and merge
  - multiscale query readout

## Use This Pack As

- archived planning material
- historical prompts for the first architecture direction

## Do Not Use It As

- the source of truth for the current repo
- the current benchmark summary
- the current module layout

For the current state, use the root [README.md](/home/dause/Desktop/TWR-LM/README.md), `configs/`, and `src/`.
